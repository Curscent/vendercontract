# Vendor Credit Flow

A Soroban-based lending system where vendors receive inventory financing instead of cash.

## Problem
Street vendors in the Philippines rely on high-interest informal loans for daily inventory.

## Solution
Smart contract releases USDC directly to suppliers and tracks repayment on-chain.

## Timeline
- Week 1: Smart contract
- Week 2: UI + demo

## Stellar Features
- USDC transfers
- Soroban smart contracts

## Vision
Enable fair financing for informal economies.

## Prerequisites
- Rust
- Soroban CLI

## Build
soroban contract build

## Test
cargo test

## Deploy
soroban contract deploy

## Sample Call
soroban contract invoke --id <ID> --fn request_loan
